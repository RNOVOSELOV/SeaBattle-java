# SeaButtle-java

РЕАЛИЗОВАНО:
1. Сделайте так, чтобы корабль располагался в случайной ячейке
2. Сделайте так, чтобы корабль был не однопалубный, а, например, трехпалубный
3. Сделайте так, чтобы кораблей было несколько!

+ Программа корректно реагирует на изменение размерности игрового поля
+ Вручную можно добавить любое колличество кораблей, любой "палубности" 

За неимением времени не стал реализовывать функционал:
- запрос количества кораблей от пользователя
- запрос на палубность кораблей от пользователя
- мультипользовательская игра
